```bash

npm run dev
# API running on http://localhost:8000

# 2. Open Postman
# Collections → Import → .ai-workflow/AdMiro_API_Postman_Collection.json

# 3. Run Health Check

# 4. Register a test user
# In Postman: Click "Register" → Send

# 5. Start testing!
# All endpoints now have your auth token automatically
```
